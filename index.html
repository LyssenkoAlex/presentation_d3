What is D3.js

D3.js is a JavaScript library for manipulating documents based on data.
D3 helps you bring data to life using HTML, SVG, and CSS.

Introduction

D3 allows you to bind arbitrary data to a Document Object Model (DOM).
D3 makes efficient manipulation of documents based on data.
D3 supports large datasets and dynamic behaviors for interaction and animation.

Selections

D3 employs a declarative approach, operating on arbitrary sets of nodes called selections. For example, you can rewrite the loop:

var paragraphs = document.getElementsByTagName("p");
for (var i = 0; i < paragraphs.length; i++) {
var paragraph = paragraphs.item(i);
paragraph.style.setProperty("color", "blue", null);
}

Converts to

d3.selectAll("p").style("color", "blue");

Enter and Exit

Using D3 enter and exit selections, you can create new nodes for incoming data.

d3.select("body")
.selectAll("p")
.data([4, 8, 15, 16, 23, 42])
.enter().append("p")
.text(function(d) { return "Got number " + d + "!"; });

6 paragraphs with Got number X

Make simple Chart

Code sample of little bar chart

var data = [30, 86, 168, 281, 303, 365];
d3.select(".chart")
.selectAll("div")
.data(data)
.enter()
.append("div")
.style("width", function(d) { return d + "px"; })
.text(function(d) { return d; });
